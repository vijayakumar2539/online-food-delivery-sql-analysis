# 🍔 Online Food Delivery Data Analysis (SQL Project)

## 📌 Project Overview
This project analyzes an online food delivery system using SQL to derive insights on customer behavior, restaurant performance, and revenue trends.

## 🛠️ Tools Used
- SQL (Joins, Aggregations, Subqueries)
- MySQL

## 📊 Key Analysis Performed
- Top revenue-generating restaurants
- Customer spending and order frequency
- Monthly order trends and peak periods
- Identification of high-value customers

## 🔍 Key Insights
- Highest orders observed in May, lowest in September
- Repeat customers contribute major revenue
- Certain restaurants consistently outperform others

## 💡 Business Recommendations
- Run promotions during low-demand months
- Introduce loyalty programs for frequent customers

## 📁 Project Structure
- `dataset/` → Raw data
- `sql_queries/` → SQL scripts
- `screenshots/` → Query outputs

## 🚀 How to Run
1. Import dataset into MySQL
2. Run SQL scripts from `sql_queries/`
3. View outputs and insights
